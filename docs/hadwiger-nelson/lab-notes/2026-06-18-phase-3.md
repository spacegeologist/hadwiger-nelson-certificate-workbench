## 529 SBP Reconstruction

The CNP-SAT repository provides `529-4-sbp.drat` but does not list a matching `529-4-sbp.cnf`. Two reconstructed symmetry-breaking variants were tested:

- `data/hadwiger-nelson/generated/529-4-sbp-add.cnf`: adds unit clauses `1 0`, `6 0`, `23 0` and updates the header from 11209 to 11212 clauses.
- `data/hadwiger-nelson/generated/529-4-sbp-replace.cnf`: replaces the relevant at-least-one color clauses with unit clauses `1 0`, `6 0`, `23 0`, keeping 11209 clauses.

Both variants verified against `529-4-sbp.drat`. The replace variant also solved quickly:

```text
variables: 2116
clauses: 11209
seconds: 2.277931
result: UNSAT
```

This suggests the published proof is valid for the strengthened SBP assumptions, and the repository likely omitted the explicit `529-4-sbp.cnf` file.
