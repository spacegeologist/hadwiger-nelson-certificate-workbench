# 2026-06-18 Phase 2

## Goal

Move from toy baseline verification to a real known 5-chromatic unit-distance graph reproduction.

## Added

- DIMACS CNF encoder for graph k-colorability.
- `cover` encoding mode: at least one color per vertex plus edge conflict clauses, omitting at-most-one clauses.
- DIMACS `p edge` parser for external graph data.
- CLI support for `--edge`, `--encoding exact`, `--encoding cover`, and `--dimacs`.
- PySAT-backed CNF solver script with timeout.
- Local `drat-trim` checkout and build.

- Heule arXiv source package `1805.12181`: contains TeX and PDF figures, but no raw graph data.
- GitHub `marijnheule/CNP-SAT`: contains raw edge files, CNF files, and DRAT proofs.
- GitHub `vsvor/dist-graphs`: contains larger DIMACS edge files and Mathematica-format vertex coordinates.

## Important Finding

Running the non-SBP `553-4.cnf` directly with PySAT/CaDiCaL timed out after 20 seconds. The SBP variant solved in about 2 seconds, and the DRAT proof verified in under 1 second.

This means the reliable reproduction path is:

1. Use raw `edge` data for graph provenance.
2. Generate or inspect cover CNF.
3. Use the published SBP CNF for fast SAT/UNSAT reproduction.
4. Use the published DRAT proof for independent certificate checking.
