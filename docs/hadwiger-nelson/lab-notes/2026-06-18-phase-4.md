# 2026-06-18 Phase 4

## Goal

Turn the reproduced CNP-SAT certificates into smaller, locally verified,
publication-style artifacts without overstating the Hadwiger-Nelson result.

## Added

- Mathematica-style `.vtx` parser in `hnp/vtx.py`.
- Embedding verification CLI in `scripts/verify_embedding.py`.
- `--preserve-edge` support in `scripts/reduce_by_core.py`.

The `.vtx` parser supports the arithmetic used by CNP-SAT coordinate files:
rationals, parentheses, multiplication, division, powers, unary signs, and
`Sqrt[...]`.

## SBP-Closed Reductions

To restore a clean graph-coloring interpretation, regenerated reductions while
preserving edges `1:2`, `1:6`, and `2:6`. These vertices correspond to SBP unit
clauses `1 0`, `6 0`, and `23 0`, i.e. vertices `1`, `2`, and `6` fixed to
three distinct colors. Because those vertices form a triangle, this is a valid
color-symmetry normalization.

## Current Claim

We have not solved the Hadwiger-Nelson problem or raised the lower bound to 6.

We do have a reproducible certificate package showing that three CNP-SAT-derived
unit-distance graphs remain non-4-colorable after removing 3, 8, and 7 edges
respectively, while keeping the symmetry-breaking support triangle and verifying
both DRAT certificates and unit-distance embeddings locally.

## Publication Assessment

This is close to a publishable computational note only if a novelty check shows
that these exact SBP-closed edge reductions, or the reproducible method of
deriving them from DRAT cores, are not already documented. The stronger 509
vertex / 2442 edge literature means the contribution is not a record-size graph.

Best positioning:

- reproducibility note,
- certificate-minimization note,
- small curated dataset with scripts and validation commands.
